Welcome to the ALMA Software Version Web Service

This service works by receiving a POST request and returning a JSON response. The POST request and Json response are as follows:

Route '/version/'

Returns the Release and Build tags on the inquired STE at the specified time, as well as all available environment information.

    Method: POST

  • Input Arguments
    • ste (required) = The STE on which you want to receive information.
    • timestamp (optional) = The DateTime on which you want to receive information. Must be a UNIX Timestamp. If not provided, the current Timestamp will be used.
    • architecture (optional) = If the inquired STE is 'AOS', this parameter indicates if the desired information comes from the 32 bit (x86) or the 64 bit (x86_64) version. The parameter is a either 32 for the 32 bit version, or 64 for the 64 bit version.
  • Output
      • Json dictionary with the following keys:
    • release = Release version used at the indicated time, at the indicated STE
    • build = Build Tag at the indicated time, at the indicated STE
    • environment = All the environment information available at the indicated time, about the indicated STE
    • ste = STE inquired
    • timestamp = Timestamp inquired
